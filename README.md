# CMDI | Community Management & Development Initiative

A sophisticated, full-stack platform designed to manage and empower community initiatives. Built with a high-performance **Next.js** frontend and a robust **PHP** backend.

---

## 🏗 Project Architecture

The project is split into two main environments:

* **/Frontend**: Next.js 14+ (App Router), powered by Turbopack for lightning-fast development.
* **/backend**: PHP-based API and service layer managed via Composer.

---
